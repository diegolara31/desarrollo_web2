Administrar mi infraestructura

Me gusta tener y administrar la infraestructura donde corren mis proyectos y servicios

Infra
Conjunto de servidores, contenedores y servicios que uso para hostear mis apps
Servicios
APIs, paginas web, bases de datos y herramientas que mantengo en mi propio ecosistema

Tareas que realizo

  • Configurar y mantener servidores (VPS o propios self-hosted)
  • Desplegar proyectos con Dokploy(Docker)
  • Monitorear servicios
  • Backups

Pasos que hago al hacer un nuevo proyecto

  1. Entrar a mi panel de control
  2. Crear mi nuevo servicio
  3. Configurar el host con GitHub para hacer deploy automatico al hacer el push
  4. Configurar el dominio
  5. Probar el servicio y verificar que funciona
  6. Configurar el monitoreo
  7. Configurar los backups
  8. Configurar la seguridad
